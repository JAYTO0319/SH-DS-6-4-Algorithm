# -신한DS-4기-6회차-알고리즘 스터디
2025년 11월 7일 개설된 알고리즘 스터디입니다.

알고리즘 스터디 기본 규칙 및 운영안

<스터디 개요>

방식: 주 1회 정기 스터디 + 개인 학습 및 피드백 공유

시간: 매주 월요일 18시~19시

장소: 강의장

방장: 김재환

부방장: 유용준, 이성훈

<기본 진행 구조>

주 1회 스터디 진행

주 단위로 주제 변경 및 문제 구성

총 5문제 풀이(개인풀이 공통 문제 3개, 실시간 풀이 문제 2개)

문제는 **방장(김재환)**이 제시하되, 참여자들의 의견을 반영하여 선정한다.

출처: 백준(BOJ), 프로그래머스 등 공개된 문제 사이트

주제와 문제 목록을 사전에 공유

<스터디 세션/1시간>

10분: 사전에 공지된(미리 풀어온) 3문제에 대해서 발표 및 토론

30분: 실시간 풀이 문제 2개는 스터디 시간에 함께 SOLVE(생성형 도구 사용 금지) 

20분: 자습(못 푼 사람은 배우고, 푼 사람은 설명 위주)
